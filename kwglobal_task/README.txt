INTRODUCTION
------------
  A Kwglobal Task created to get a user name and log message


REQUIREMENTS
------------

  No special requirements.


INSTALLATION
------------

  * follow the following steps for installation
    - go to /admin/modules 
    - find Kwglobal Task module using filter 
    - check checkbox and click on install


PERMISSION
-------------
  * go to admin/people/permissions
  * Find group Kwglobal Task
  * add permission role wise

ACCESS FORM
-------------
  * The menu will be enabled in the Tool block. Menu name: Task
  * URL: /kwglobal-task access for directly
